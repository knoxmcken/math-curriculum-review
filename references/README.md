# Reference Notebooks Summary

This directory contains CrewAI tutorial notebooks covering multi-agent AI systems.

## Notebooks Overview

### L3_customer_support.ipynb (109 KB)
**Multi-agent Customer Support Automation**

Topics covered:
- Role Playing
- Focus
- Tools
- Cooperation
- Guardrails
- Memory

Learn how to build agents that work together to provide automated customer support.

### L4_tools_customer_outreach.ipynb (109 KB)
**Tools for a Customer Outreach Campaign**

Key elements:
- Versatility
- Fault Tolerance
- Caching

Deep dive into using tools effectively within your CrewAI agents.

### L5_tasks_event_planning.ipynb (13 KB)
**Automate Event Planning**

Focus: Understanding and working with Tasks in CrewAI framework.

### L6_collaboration_financial_analysis.ipynb (13 KB)
**Multi-agent Collaboration for Financial Analysis**

Learn techniques for making agents collaborate effectively with each other.

### L7_job_application_crew.ipynb (16 KB)
**Build a Crew to Tailor Job Applications**

Build your first complete multi-agent system from scratch.
